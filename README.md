# Algo C2

Experimental trading research and training code for the `staged_v4` pipeline.

Current repo scope:
- model training and replay code
- data preparation and collector tooling
- remote run helpers and operational scripts

Runtime layout:
- local machine: code and analysis
- collector machine: MT5-backed data collection
- training machine / RunPod: model training only

This repository keeps the git-facing overview intentionally brief.
